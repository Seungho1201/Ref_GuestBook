## 팀원
| 이름 | 역할 |
| :---: | :---: |
| 오승호 | <b>팀장</b>, 기본 그리기, 리플레이 스레드, 펜 굵기 조절, 버튼 클래스 분할, 스테이터스 기능 |
| 김용빈 | 파일 세이브 및 로드, 지우기 기능 |
| 용지순 | 스티커 기능 |
| 임지섭 | 그리기 영역, 저장된 전체 파일 재생 기능 |
| 전성호 | 윈도우 크기 고정 |
| 장환희 | 펜 색상 변경, 펜 미리보기 기능 |
